# PTG Gaming Survey (Option A — Subfolder Deployment)

✅ Deploy path:
`/PTG1/gaming-survey/`

✅ Live URL:
`https://professorzoom45.github.io/PTG1/gaming-survey/`

## Files
- `index.html` — the survey page (PTG theme + images + market research questions)
- `styles.css` — neon chalkboard theme
- `script.js` — validation, auto-save, and email submit builder
- `ptg_logo.png` — PTG logo image
- `PTG MISSION AND RULES.png` — Mission & Rules poster image

## How to upload
1. In your GitHub repo `PTG1`, create a folder named `gaming-survey`
2. Upload **all files** in this folder into that `gaming-survey/` folder
3. Wait 1–3 minutes for GitHub Pages to update
4. Open the live URL above

## Notes
- Email submit uses `mailto:` to `changethewrld@outlook.com`
- If a device blocks mailto, use the built-in **Copy Answers** button and paste into an email.
